# ☁️ BlogDePruebaAWS

Aplicación **Ruby on Rails** usada como laboratorio de despliegue y aprendizaje en AWS, acompañada de documentación técnica sobre servicios cloud.

![Rails](https://img.shields.io/badge/Rails-app-CC0000?logo=rubyonrails) ![AWS](https://img.shields.io/badge/AWS-learning-FF9900?logo=amazonaws)

## Descripción

Incluye una app Rails con estructura convencional, archivos de despliegue, CI y una serie de documentos Markdown dedicados a IAM, S3, EC2 y RDS. El repositorio mezcla desarrollo web y aprendizaje de infraestructura.

## Objetivo

Usar un blog de prueba como base para practicar despliegue, configuración de servicios AWS y documentación técnica aplicada.

## Uso

```bash
bundle install
bin/rails db:setup
bin/rails server
```
